## ✨ Key Features

*   **🚀 Git-Powered Core**: Leverages the rock-solid foundation of Git for all file storage and versioning. Snapshots are lightweight, annotated Git tags pointing to specific commits.
*   **💨 Blazing Fast & Scalable**: Designed for performance on projects of all sizes. `snap` uses a hybrid approach: native `git.exe` for filesystem-heavy tasks (like checking for changes) and a lightweight internal library for simple reads, ensuring it's fast on both small and multi-gigabyte repositories.
*   **⚡️ Efficient Storage**: Benefits from Git's mature Content-Addressable Storage model. Identical files across hundreds of snapshots consume the space of just one file.
*   **🎯 Smart Restores**: `snap restore` intelligently checks for uncommitted local changes and prompts you to prevent accidental data loss. It also restores your project files and synchronizes hidden file attributes.
*   **🪟 Windows Native**: Fully aware of Windows file attributes. It correctly preserves and restores the **Hidden** status of files and folders using an external metadata store.
*   **✍️ Rich Metadata**: Snapshots are tagged with a simple `label` and a more detailed `description`, which are stored directly in Git's annotated tags.
*   **💻 User-Friendly CLI**: Features an interactive, arrow-key menu, human-readable timestamps, a compact list view, and an **`(active)`** marker to show you exactly which version your project is on.
*   **🚫 Zero Dependencies (for users)**: Once built, the `snap.exe` tool runs without requiring Node.js to be installed on the user's machine.

## ⚙️ How It Works

`snap` uses a clever hybrid approach. It manages your project as a standard Git repository but interacts with it in two different ways to maximize performance and reliability.

1.  **The Project Repository (`.git`)**
    When you run `snap init`, your project folder becomes a standard Git repository.

2.  **The Interaction Model (Fast Reads, Robust Writes)**
    *   **For Fast Reads (e.g., getting the active commit):** `snap` uses an internal, pure JavaScript Git library (`isomorphic-git`) to read simple data from the `.git` folder. This is extremely fast for targeted reads.
    *   **For Robust Reads & Writes (e.g., `list`, `new`, `restore`):** For most operations, especially those that are complex or filesystem-heavy, `snap` calls the official `git.exe` command-line tool. This ensures maximum compatibility, speed, and robustness.

3.  **The External Metadata Store**
    Because Git doesn't track the "Hidden" attribute, `snap` saves this information in a central `backup_root` directory you configure. For each snapshot, it creates a simple JSON file listing all hidden paths.

**Visualized Structure:**

```text
// Your project folder becomes a Git repository:
D:/Projects/MyUnityGame/
└── .git/

// A separate, central location stores the hidden attribute metadata:
D:/Backups/Snap_Metadata/
└── MyUnityGame_[a4e8b91c]/  <-- Human-readable, unique ID for your project
    ├── 0a1b2c...d3e4.json   <-- Hidden paths for commit 0a1b...
    └── 5f6a7b...c8d9.json   <-- Hidden paths for commit 5f6a...
```

This design keeps your project folder clean and standard while ensuring no data is lost and performance is optimal.

## 📥 Installation

Installation involves a one-time build step to create the fast, single executable.

1.  **Get the Project Files**
    Clone or download the repository to a permanent location on your system (e.g., `D:\Tools\snap-tool`). The main project folder is `snap_bat`.

2.  **Install Dependencies & Build**
    Open a command prompt and navigate to the **`snap_bat`** directory. Run the following commands:
    ```cmd
    cd D:\Tools\snap-tool\snap_bat
    
    rem This installs dependencies like inquirer, chalk, isomorphic-git, etc.
    npm install
    
    rem This builds the single, fast, standalone snap.exe file.
    npm run build
    ```
    After this step, you will see a new `snap.exe` file inside the `snap_bat` folder. This is your program.

3.  **Update Your PATH**
    Add the `snap_bat` directory (e.g., `D:\Tools\snap-tool\snap_bat`) to your Windows PATH environment variable. This allows you to run the `snap` command from any terminal.

4.  **First-Time Configuration**
    The first time you run `snap init` in a project, it will prompt you to set up a central `backup_root` directory. This is where the metadata for hidden files will be stored. This configuration is saved in a `.snapconfig` file alongside your `snap.exe`.

5.  **Verify**
    Open a **new** terminal window (so it loads the new PATH). Navigate to a project you want to back up and run `snap init`. It should initialize the repository.

## 🚀 Usage

All commands are run from within your project's directory.

### 1. `snap init`
Initializes the current folder as a `snap` repository. **This must be run once for each project.**

```cmd
D:\Projects\my-app>snap init
[snap] The `backup_root` location... is not configured.
? Enter the full path for your central backup directory: D:/Snap_Metadata
[snap] Set backup_root to: D:/Snap_Metadata
[snap] Initialized empty snap repository in D:\Projects\my-app
```

### 2. `snap new [<label>] [<description...>]`
Creates a snapshot of the current project state by committing all changes and creating an annotated Git tag.

```cmd
D:\Projects\my-app>snap new v1.0 "Initial release"

[snap] Step 1/4: Staging all files...
[snap] Step 2/4: Creating the commit...
[snap] Step 3/4: Saving file attribute metadata...
[snap] Step 4/4: Creating the annotated snapshot tag...

[snap] New snapshot created: [a1b2c3d] v1.0
```

### 3. `snap list`
Lists all available snapshots in a compact view, showing the active one.

```cmd
D:\Projects\my-app>snap list
[snap] Snapshots for "my-app":

  Label           Description                 Timestamp
  --------------  --------------------------  ----------------
  v1.1-hotfix     Fixed critical login bug    15.06.2025 10:30
  v1.1            Second public release       15.06.2025 09:15   (active)
  v1.0            Initial release             15.06.2025 17:45
```

### 4. `snap diff <snapshot-A> <snapshot-B>`
Compares two snapshots and shows a list of added, deleted, and modified files, as well as changes in file visibility (Hidden attribute).

```cmd
D:\Projects\my-app>snap diff v1.0 v1.1
[snap] Comparing snapshots a1b2c3d ("v1.0") ➜ f4e5d6c ("v1.1"):

  + src/auth/new-logic.js
  - config/old-settings.json
  ~ src/app.js
  ! .env (visibility changed)

[snap] Summary: 1 added, 1 deleted, 1 modified, 1 visibility change.
```

### 5. `snap restore [<id_or_label>]`
Restores the project to a previous state. It checks out the corresponding Git tag and synchronizes the hidden file attributes. If run without arguments, it displays an interactive menu.

```cmd
D:\Projects\my-app>snap restore v1.0
[snap] Restoring project files for snapshot "v1.0"...
[snap] Synchronizing hidden file attributes...
[snap] Restore complete. Your project is now at the state of this snapshot.
```

### 6. `snap delete [<id_or_label>]`
Permanently deletes a snapshot. This removes the Git tag and its associated hidden attribute metadata file.

```cmd
D:\Projects\my-app>snap delete v1.1-hotfix
[snap] You are about to delete snapshot:
  Label: v1.1-hotfix
[snap] WARNING: This will permanently delete the snapshot tag. Continue? [y/N] y
[snap] Deleting tag "v1.1-hotfix"...
[snap] Cleaned up attribute metadata.
[snap] Snapshot "v1.1-hotfix" deleted successfully.
```

### 7. `snap edit [<id_or_label>]`
Edits the label (tag name) and description of an existing snapshot.

```cmd
D:\Projects\my-app>snap edit v1.0
[snap] Editing snapshot "v1.0":
? Enter new label (tag name): v1.0-final
? Enter new description: Final version for initial release
...
[snap] Snapshot successfully updated to "v1.0-final".
```

### 8. `snap update`
Amends the **active** snapshot with the current state of the project. This is for making small fixes to the last snapshot without creating a whole new one. It amends the commit, updates the hidden attribute metadata, and moves the Git tag.

```cmd
D:\Projects\my-app>snap update
[snap] This command will replace the active snapshot...
  Target Snapshot:
    Label:       v1.1
[snap] This will amend the commit... This action is hard to undo. [y/N] y
...
[snap] Update complete. Snapshot "v1.1" now points to new commit [b8c9d0e].
```

### 9. `snap options`
Allows you to configure global settings for the tool, such as whether to display snapshot IDs.

```cmd
D:\Projects\my-app>snap options
? Select option to change: (Use arrow keys)
> showIds         - (Controls if IDs are shown in lists)
  confirm_command - (Asks for y/N on destructive actions)
```

## 🛠️ Troubleshooting

*   **`Error: Git is not installed or not in your system PATH.`**: This can still appear if a command like `init` or `restore` fails. `snap` requires Git for its core operations. Install Git for Windows and ensure its `bin` and `cmd` directories are in your system's PATH.
*   **`'snap' is not recognized...`**: This means the `snap_bat` directory was not correctly added to your PATH, or you haven't opened a new terminal window since adding it.
*   **`Error: Not a snap repository...`**: You are trying to run a command (like `list` or `new`) inside a directory that has not been initialized. Run `snap init` first.
*   **`Error: EPERM: operation not permitted, unlink '...snap.exe'`**: The `snap.exe` executable is locked. Close all terminals where you might have run `snap`, or use Task Manager to ensure no `snap.exe` process is running, then run `npm run build` again.

#!/usr/bin/env node
const fs = require('fs');
const path = require('path');

const chalk = {
    cyan: (str) => `\x1b[36m${str}\x1b[0m`,
    yellow: (str) => `\x1b[33m${str}\x1b[0m`,
    red: (str) => `\x1b[31m${str}\x1b[0m`,
    green: (str) => `\x1b[32m${str}\x1b[0m`,
    gray: (str) => `\x1b[90m${str}\x1b[0m`,
    bold: (str) => `\x1b[1m${str}\x1b[22m`,
};

const SNAP_EXE_DIR = path.dirname(process.execPath);
const VIRTUAL_DIR = __dirname;
const CMDS_DIR_VIRTUAL = path.join(VIRTUAL_DIR, 'snap_cmds');
const CONFIG_PATH = path.join(SNAP_EXE_DIR, '.snapconfig');

// The synchronous git check has been removed to improve startup speed.
// Individual commands will now perform this check asynchronously.

function getAvailableCommands() {
    try {
        return fs.readdirSync(CMDS_DIR_VIRTUAL)
            .filter(file => file.endsWith('.js') && file !== 'utils.js')
            .map(file => path.parse(file).name);
    } catch (e) { return []; }
}

function showHelp() {
    console.log('Usage: snap <command> [options]');
    console.log('\nAvailable snap commands:');
    const commands = getAvailableCommands().sort();
    for (const cmdName of commands) {
        const descPath = path.join(CMDS_DIR_VIRTUAL, `${cmdName}.desc`);
        let desc = 'No description.';
        if (fs.existsSync(descPath)) {
            desc = fs.readFileSync(descPath, 'utf8').trim();
        }
        console.log(`  ${chalk.bold(cmdName.padEnd(10))} ${desc}`);
    }
    console.log(`\nGlobal options:\n  --debug    Enable detailed diagnostic output.`);
}

(async function main() {
    let args = process.argv.slice(2);
    
    // ** DEBUG FLAG HANDLING **
    const debugIndex = args.indexOf('--debug');
    if (debugIndex > -1) {
        process.env.SNAP_DEBUG = 'true';
        args.splice(debugIndex, 1); // Remove the flag from args
        console.log(chalk.yellow.bold('--- DEBUG MODE ENABLED ---'));
    }

    const command = args[0];

    if (!command) {
        showHelp();
        process.exit(0);
    }
    
    if (!fs.existsSync(CONFIG_PATH) && command !== 'init') {
        console.error(chalk.red(`[snap] ERROR: Configuration file ".snapconfig" not found.`));
        process.exit(1);
    }
    
    process.env.SNAP_CONFIG_PATH = CONFIG_PATH;

    const cmdScriptPath = path.join(CMDS_DIR_VIRTUAL, `${command}.js`);

    if (!fs.existsSync(cmdScriptPath)) {
        console.error(chalk.red(`[snap] Unknown command: "${command}"`));
        showHelp();
        process.exit(1);
    }
    
    try {
        const commandModule = require(cmdScriptPath);
        const subArgs = args.slice(1);
        await commandModule(subArgs);
    } catch (error) {
        console.error(chalk.red(`\n[snap] An unexpected error occurred while running the "${command}" command:`));
        console.error(error.message);
        if (error.stderr) {
            console.error(chalk.gray('--- STDERR ---'));
            console.error(chalk.gray(error.stderr));
            console.error(chalk.gray('--------------'));
        }
        process.exit(1);
    }
})();

const inquirer = require('inquirer');
const {
    isGitRepo, getSnapshots, findSnapshot, askYesNo,
    runCommand, chalk, formatSnapshotLine, getConfig,
    deleteExternalMetadata
} = require('./utils');

module.exports = async (args) => {
    if (!(await isGitRepo())) {
        console.error(chalk.red('[snap] Error: Not a snap repository. Run "snap init" first.'));
        process.exit(1);
    }

    let idOrLabel = args[0];
    const snapshots = await getSnapshots();

    if (snapshots.length === 0) {
        console.error(chalk.yellow('[snap] No snapshots found to delete.'));
        process.exit(1);
    }

    if (!idOrLabel) {
        const { selectedTag } = await inquirer.prompt([{
            type: 'list',
            name: 'selectedTag',
            message: 'Select snapshot to delete:',
            choices: snapshots.map(s => ({ name: formatSnapshotLine(s), value: s.tag })),
            pageSize: 15,
        }]);
        idOrLabel = selectedTag;
    }

    const snapshotInfo = findSnapshot(snapshots, idOrLabel);
    if (!snapshotInfo) {
        console.error(chalk.red(`[snap] Error: Snapshot "${idOrLabel}" not found.`));
        process.exit(1);
    }
    
    console.log(chalk.cyan('\n[snap] You are about to delete snapshot:'));
    console.log(`  Label: ${snapshotInfo.tag}`);
    const { confirm_command } = getConfig().options;
    if (confirm_command && !(await askYesNo('[snap] WARNING: This will permanently delete the snapshot tag. Continue?'))) {
        console.log(chalk.yellow('[snap] Deletion cancelled.'));
        return;
    }

    console.log(chalk.cyan(`[snap] Deleting tag "${snapshotInfo.tag}"...`));
    runCommand(`git tag -d ${snapshotInfo.tag}`);

    deleteExternalMetadata(snapshotInfo.fullId);

    console.log(chalk.green(`[snap] Snapshot "${snapshotInfo.tag}" deleted successfully.`));
};

snap_cmds/diff.js:
```javascript
const { isGitRepo, runCommand, findSnapshot, getSnapshots, chalk, loadExternalMetadata } = require('./utils');

const statusMap = {
    'A': { prefix: '+', color: 'green', label: 'added' },
    'D': { prefix: '-', color: 'red', label: 'deleted' },
    'M': { prefix: '~', color: 'yellow', label: 'modified' },
    'R': { prefix: '»', color: 'cyan', label: 'renamed' },
    'C': { prefix: 'C', color: 'cyan', label: 'copied' },
};

module.exports = async (args) => {
    if (!(await isGitRepo())) {
        console.error(chalk.red('[snap] Error: Not a snap repository. Run "snap init" first.'));
        process.exit(1);
    }

    if (args.length !== 2) {
        console.error(chalk.red('[snap] Error: The "diff" command requires exactly two snapshot identifiers.'));
        console.error(chalk.gray('  Usage: snap diff <snapshot-A> <snapshot-B>'));
        process.exit(1);
    }

    const [idOrLabelA, idOrLabelB] = args;
    const snapshots = await getSnapshots();
    const snapshotA = findSnapshot(snapshots, idOrLabelA);
    const snapshotB = findSnapshot(snapshots, idOrLabelB);

    if (!snapshotA || !snapshotB) {
        console.error(chalk.red(`[snap] Error: Could not find one or both snapshots.`));
        process.exit(1);
    }

    console.log(chalk.cyan(`\n[snap] Comparing snapshots ${snapshotA.id} ("${snapshotA.tag}") ➜ ${snapshotB.id} ("${snapshotB.tag}"):\n`));

    const hiddenA = new Set(loadExternalMetadata(snapshotA.fullId, 'hidden'));
    const hiddenB = new Set(loadExternalMetadata(snapshotB.fullId, 'hidden'));
    const emptyA = new Set(loadExternalMetadata(snapshotA.fullId, 'emptydirs'));
    const emptyB = new Set(loadExternalMetadata(snapshotB.fullId, 'emptydirs'));

    const diffOutput = runCommand(`git diff --name-status ${snapshotA.fullId} ${snapshotB.fullId}`);
    const changes = { added: 0, deleted: 0, modified: 0, renamed: 0, copied: 0, visibility: 0, empty_added: 0, empty_removed: 0 };
    const allFilesMentioned = new Set();

    if (diffOutput) {
        for (const line of diffOutput.split('\n')) {
            const [status, file, newFile] = line.split('\t');
            const statusChar = status.charAt(0);
            const mapEntry = statusMap[statusChar] || { prefix: '?', color: 'gray', label: 'unknown' };
            console.log(chalk[mapEntry.color](`  ${mapEntry.prefix} ${file}${newFile ? ` -> ${newFile}` : ''}`));
            allFilesMentioned.add(file.replace(/\\/g, '/'));
            if (newFile) allFilesMentioned.add(newFile.replace(/\\/g, '/'));
            if (mapEntry.label in changes) changes[mapEntry.label]++;
        }
    }

    const allHiddenPaths = new Set([...hiddenA, ...hiddenB]);
    for (const filePath of allHiddenPaths) {
        if (allFilesMentioned.has(filePath)) continue;
        if (hiddenA.has(filePath) !== hiddenB.has(filePath)) {
            console.log(chalk.magenta(`  ! ${filePath} (visibility changed)`));
            changes.visibility++;
        }
    }

    for (const dir of emptyB) {
        if (!emptyA.has(dir)) {
            console.log(chalk.green(`  + ${dir}/ (empty directory)`));
            changes.empty_added++;
        }
    }
    for (const dir of emptyA) {
        if (!emptyB.has(dir)) {
            console.log(chalk.red(`  - ${dir}/ (empty directory)`));
            changes.empty_removed++;
        }
    }

    const summaryParts = Object.entries(changes)
        .filter(([, count]) => count > 0)
        .map(([label, count]) => {
            if (label === 'visibility') return `${count} visibility ${count > 1 ? 'changes' : 'change'}`;
            if (label === 'empty_added') return `${count} empty dir added`;
            if (label === 'empty_removed') return `${count} empty dir removed`;
            return `${count} ${label}`;
        });

    if (summaryParts.length === 0) {
        console.log(chalk.green('[snap] The two snapshots are identical. No differences found.'));
    } else {
        console.log(chalk.cyan(`\n[snap] Summary: ${summaryParts.join(', ')}.`));
    }
};

const inquirer = require('inquirer');
const {
  isGitRepo, getSnapshots, findSnapshot, runCommand, chalk, formatSnapshotLine, logDebug
} = require('./utils');

function sanitizeTagName(label) {
    if (!label) return null;
    return label.trim().replace(/\s+/g, '-').replace(/[^a-zA-Z0-9-._]/g, '');
}

module.exports = async (args) => {
  logDebug("Executing 'edit' command...");
  if (!(await isGitRepo())) {
    console.error(chalk.red('[snap] Error: Not a snap repository. Run "snap init" first.'));
    process.exit(1);
  }

  let idOrLabel = args[0];
  const snapshots = await getSnapshots();

  if (snapshots.length === 0) {
    console.error(chalk.yellow('[snap] No snapshots found to edit.'));
    process.exit(1);
  }

  if (!idOrLabel) {
    const { selectedTag } = await inquirer.prompt([{
      type: 'list',
      name: 'selectedTag',
      message: 'Select snapshot to edit:',
      choices: snapshots.map(s => ({ name: formatSnapshotLine(s), value: s.tag })),
      pageSize: 15,
    }]);
    idOrLabel = selectedTag;
  }

  const snap = findSnapshot(snapshots, idOrLabel);
  if (!snap) {
    console.error(chalk.red(`[snap] Error: Snapshot "${idOrLabel}" not found.`));
    process.exit(1);
  }

  console.log(chalk.cyan(`\n[snap] Editing snapshot "${snap.tag}":`));
  
  const { newLabel, newDescription } = await inquirer.prompt([
    {
      type: 'input',
      name: 'newLabel',
      message: 'Enter new label (tag name):',
      default: snap.tag,
      validate: v => !!v.trim() || 'Label cannot be empty.',
    },
    {
      type: 'input',
      name: 'newDescription',
      message: 'Enter new description:',
      default: snap.description,
    }
  ]);

  const newTagName = sanitizeTagName(newLabel);
  const newTagMessage = newDescription.trim() || newTagName;

  if (snap.tag === newTagName && snap.description === newTagMessage) {
    console.log(chalk.yellow('[snap] No changes detected. Edit cancelled.'));
    return;
  }
  
  console.log(chalk.cyan('\n[snap] Review changes:'));
  console.log(`  Old Label:       ${snap.tag}`);
  console.log(`  New Label:       ${newTagName}`);
  console.log(`  Old Description: ${snap.description}`);
  console.log(`  New Description: ${newTagMessage}`);
  
  console.log(chalk.yellow(`\n[snap] Applying changes... This will replace the old tag.`));

  // Tagging is a write operation, `runCommand` is fine.
  runCommand(`git tag -d "${snap.tag}"`, `Deleting old tag "${snap.tag}"`);
  runCommand(`git tag -a "${newTagName}" -m "${newTagMessage}" ${snap.fullId}`, `Creating new tag "${newTagName}" on commit ${snap.id}`);

  console.log(chalk.green(`\n[snap] Snapshot successfully updated to "${newTagName}".`));
};

const fs = require('fs');
const path = require('path');
const inquirer = require('inquirer');
const { runCommand, isGitRepo, chalk, getConfig, updateConfigOption } = require('./utils');

module.exports = async (args) => {
    if (await isGitRepo()) {
        console.error(chalk.yellow('[snap] This directory is already a Git repository.'));
        return;
    }

    let config = getConfig();
    if (!config.backup_root || !fs.existsSync(config.backup_root)) {
        console.log(chalk.yellow('[snap] The `backup_root` location for storing file attribute metadata is not configured.'));
        const { backupRootPath } = await inquirer.prompt([{
            type: 'input',
            name: 'backupRootPath',
            message: 'Enter the full path for your central backup directory:',
            validate: (input) => {
                if (!input) return 'Path cannot be empty.';
                if (!path.isAbsolute(input)) return 'Please provide an absolute path (e.g., D:/SnapBackups).';
                return true;
            }
        }]);

        const normalizedPath = backupRootPath.replace(/\\/g, '/');
        try {
            fs.mkdirSync(normalizedPath, { recursive: true });
            updateConfigOption('backup_root', normalizedPath);
            console.log(chalk.green(`[snap] Set backup_root to: ${normalizedPath}`));
        } catch (e) {
            console.error(chalk.red(`[snap] Error: Could not create or access directory: ${normalizedPath}`));
            console.error(chalk.red(e.message));
            process.exit(1);
        }
    }

    if (!process.env.SNAP_CONFIG_PATH || !fs.existsSync(process.env.SNAP_CONFIG_PATH)) {
        updateConfigOption('showIds', false);
        updateConfigOption('confirm_command', true);
        console.log(chalk.gray('[snap] Created default .snapconfig file with default options.'));
    }
    
    // `git init` is a one-time setup command, `runCommand` is perfect.
    runCommand('git init');
    console.log(chalk.green(`[snap] Initialized empty snap repository in ${process.cwd()}`));
};

function sanitizeTagName(label) {
    if (!label) return `snap-${Date.now()}`;
    return label.trim().replace(/\s+/g, '-').replace(/[^a-zA-Z0-9-._]/g, '');

module.exports = async (args) => {
    logDebug("Executing 'new' command...");

        process.exit(1);
    }

    const label = args[0];
    if (!label) {
        console.error(chalk.red('[snap] Error: A label is required for a new snapshot.'));
        console.error(chalk.gray('  Usage: snap new <label> [<description...>]'));

    const tagName = sanitizeTagName(label);

    // ✅ FIX: Check for duplicate tag name BEFORE doing any work.
    const existingSnapshots = await getSnapshots();
    if (existingSnapshots.some(s => s.tag === tagName)) {
        console.error(chalk.red(`[snap] Error: A snapshot with the label "${tagName}" already exists.`));
        console.error(chalk.gray('  Please choose a unique label or delete the existing snapshot first.'));
        process.exit(1); // Abort cleanly.
    }

    // Now that the tag is validated, check for changes.
    if (!(await checkDirty())) {
        console.log(chalk.yellow('[snap] No changes to commit. Working tree is clean.'));
        return;
    }

    const description = args.slice(1).join(' ').trim();

    console.log(chalk.green('\n[snap] Step 1/4: Staging all files...'));
    runCommand('git add -A', 'Staging all changes');

    console.log(chalk.green('[snap] Step 2/4: Creating the commit...'));
    const commitMessage = `Snapshot: ${tagName}`;
    runCommand(`git commit -m "${commitMessage}"`, 'Creating commit');

    const fullCommitId = await getActiveCommitFull();

    // This correctly calls your custom function.
    console.log(chalk.green('[snap] Step 3/4: Saving external metadata (hidden files, empty dirs)...'));
    await saveExternalMetadata(fullCommitId);

    console.log(chalk.green('[snap] Step 4/4: Creating the annotated snapshot tag...'));

    const tagMessage = description;
    const command = `git tag -a "${tagName}" -F -`;
    const cmdDescription = `Creating annotated tag ${tagName}`;
    
    runCommand(command, cmdDescription, { input: tagMessage });

    const shortCommitId = fullCommitId.substring(0, 7);
    console.log(chalk.green(`\n[snap] New snapshot created: [${shortCommitId}] ${tagName}`));
};

```javascript
const fs = require('fs');
const inquirer = require('inquirer');
const path = require('path');
const {
    isGitRepo, getSnapshots, findSnapshot, askYesNo, checkDirty,
    runCommand, chalk, formatSnapshotLine,
    getActiveCommitFull, loadExternalMetadata, logDebug
} = require('./utils');

module.exports = async (args) => {
    if (!(await isGitRepo())) {
        console.error(chalk.red('[snap] Error: Not a snap repository. Run "snap init" first.'));
        process.exit(1);
    }

    const sourceCommitId = await getActiveCommitFull();

    if (await checkDirty()) {
        console.log(chalk.yellow('\n[snap] WARNING: Your project has uncommitted changes.'));
        const proceed = await askYesNo('To restore a snapshot, all local changes must be discarded. Continue?');
        if (!proceed) {
            console.log(chalk.yellow('[snap] Restore cancelled.'));
            process.exit(0);
        }
        console.log(chalk.cyan('[snap] Discarding all local changes...'));
        runCommand('git reset --hard HEAD');
        runCommand('git clean -fd');
        console.log(chalk.green('[snap] Workspace is now clean. Proceeding with restore.\n'));
    }

    let idOrLabel = args[0];
    const snapshots = await getSnapshots();
    if (snapshots.length === 0) {
        console.error(chalk.yellow('[snap] No snapshots found to restore.'));
        process.exit(1);
    }

    if (!idOrLabel) {
        const { selectedTag } = await inquirer.prompt([{
            type: 'list',
            name: 'selectedTag',
            message: 'Select snapshot to restore:',
            choices: snapshots.map(s => ({ name: formatSnapshotLine(s), value: s.tag })),
            pageSize: 15,
        }]);
        idOrLabel = selectedTag;
    }

    const snapshotInfo = findSnapshot(snapshots, idOrLabel);
    if (!snapshotInfo) {
        console.error(chalk.red(`[snap] Error: Snapshot "${idOrLabel}" not found.`));
        process.exit(1);
    }

    console.log(chalk.cyan(`\n[snap] Restoring project files for snapshot "${snapshotInfo.tag}"...`));
    runCommand(`git checkout --force ${snapshotInfo.tag}`);

    console.log(chalk.cyan('[snap] Synchronizing external metadata...'));
    
    // Load all metadata types
    const sourceHidden = loadExternalMetadata(sourceCommitId, 'hidden');
    const targetHidden = loadExternalMetadata(snapshotInfo.fullId, 'hidden');
    const sourceEmpty = loadExternalMetadata(sourceCommitId, 'emptydirs');
    const targetEmpty = loadExternalMetadata(snapshotInfo.fullId, 'emptydirs');
    
    const sourceHiddenSet = new Set(sourceHidden);
    const targetHiddenSet = new Set(targetHidden);
    const sourceEmptySet = new Set(sourceEmpty);
    const targetEmptySet = new Set(targetEmpty);

    // Reconcile empty directories
    logDebug('Reconciling empty directories...');
    const dirsToCreate = targetEmpty.filter(p => !sourceEmptySet.has(p) && !fs.existsSync(path.join(process.cwd(), p)));
    const dirsToRemove = sourceEmpty.filter(p => !targetEmptySet.has(p) && fs.existsSync(path.join(process.cwd(), p)));
    dirsToRemove.sort((a, b) => b.length - a.length); // Remove children before parents

    for (const relativePath of dirsToRemove) {
        const fullPath = path.join(process.cwd(), relativePath);
        try {
            if (fs.readdirSync(fullPath).length === 0) {
                logDebug(`  - Removing empty directory: ${relativePath}`);
                fs.rmdirSync(fullPath);
            }
        } catch (e) { logDebug(`Failed to remove directory ${relativePath}: ${e.message}`); }
    }
    for (const relativePath of dirsToCreate) {
        const fullPath = path.join(process.cwd(), relativePath);
        logDebug(`  - Creating empty directory: ${relativePath}`);
        try { fs.mkdirSync(fullPath, { recursive: true }); } 
        catch (e) { logDebug(`Failed to create directory ${relativePath}: ${e.message}`); }
    }

    // Reconcile hidden file attributes
    logDebug('Reconciling hidden file attributes...');
    const pathsToUnhide = sourceHidden.filter(p => !targetHiddenSet.has(p) && fs.existsSync(p));
    const pathsToHide = targetHidden.filter(p => fs.existsSync(p));

    const applyAttribute = (verb, attr, paths) => {
        for (const relativePath of paths) {
            const fullPath = path.join(process.cwd(), relativePath);
            logDebug(`  - Making ${verb} (${attr}): ${relativePath}`);
            try { runCommand(`attrib ${attr} "${fullPath}"`); } catch (e) {
                logDebug(`Failed to apply attribute to ${relativePath}: ${e.message}`);
            }
        }
    };
    
    applyAttribute('visible', '-H', pathsToUnhide);
    applyAttribute('hidden', '+H', pathsToHide);

    console.log(chalk.green('\n[snap] Restore complete. Your project is now at the state of this snapshot.'));
};

const {
    isGitRepo, getSnapshots, getActiveCommitFull, runCommand,
    askYesNo, checkDirty, chalk, getConfig, findSnapshot,
    deleteExternalMetadata, saveExternalMetadata
} = require('./utils');

module.exports = async (args) => {
    if (!(await isGitRepo())) {
        console.error(chalk.red('[snap] Error: Not a snap repository. Run "snap init" first.'));
        process.exit(1);
    }

    if (!(await checkDirty())) {
        console.log(chalk.yellow('[snap] No changes to update. Working tree is clean.'));
        return;
    }

    const activeCommitFull = await getActiveCommitFull();
    if (!activeCommitFull) {
        console.error(chalk.red('[snap] Error: Cannot update. No active snapshot found.'));
        process.exit(1);
    }

    const snapshots = await getSnapshots();
    const activeSnapshot = findSnapshot(snapshots, activeCommitFull);
    if (!activeSnapshot) {
        console.error(chalk.red('[snap] Error: The current state (HEAD) does not correspond to a known snapshot.'));
        console.error(chalk.gray('[snap] Hint: You may need to run `snap restore` to an existing snapshot first.'));
        process.exit(1);
    }

    console.log(chalk.cyan(`\n[snap] This command will replace the active snapshot with the current project state.`));
    console.log(chalk.yellow('  Target Snapshot:'));
    console.log(`    Label:       ${activeSnapshot.tag}`);
    console.log(`    Description: ${activeSnapshot.description}`);


    const { confirm_command } = getConfig().options;
    const proceed = confirm_command ? await askYesNo(`[snap] This will amend the commit for snapshot "${activeSnapshot.tag}". This action is hard to undo.`) : true;
    
    if (!proceed) {
        console.log(chalk.yellow('[snap] Update cancelled.'));
        return;
    }

    console.log(chalk.cyan('\n[snap] Step 1/4: Deleting old external metadata...'));
    deleteExternalMetadata(activeSnapshot.fullId);

    console.log(chalk.cyan('[snap] Step 2/4: Staging changes and amending commit...'));
    runCommand('git add -A', 'Staging changes');
    runCommand('git commit --amend --no-edit --allow-empty', 'Amending commit');

    const newCommitId = await getActiveCommitFull();

    console.log(chalk.cyan('[snap] Step 3/4: Saving new external metadata...'));
    await saveExternalMetadata(newCommitId);

    const newShortCommitId = newCommitId.substring(0, 7);
    console.log(chalk.cyan(`[snap] Step 4/4: Moving tag "${activeSnapshot.tag}" to new commit ${newShortCommitId}...`));
    
    const tagMessage = activeSnapshot.description;
    const command = `git tag -a -f "${activeSnapshot.tag}" -F -`;
    const description = `Re-creating annotated tag ${activeSnapshot.tag}`;
    
    runCommand(command, description, { input: tagMessage });

    console.log(chalk.green(`\n[snap] Update complete. Snapshot "${activeSnapshot.tag}" now points to new commit [${newShortCommitId}].`));
};

snap_cmds/utils.js:
```javascript
const fs = require('fs');
const path = require('path');
const crypto = require('crypto');
const { execSync } = require('child_process');
const readline = require('readline');
const chalk = require('chalk');
const git = require('isomorphic-git');

// ** DEBUG HELPER **
function logDebug(message) {
    if (process.env.SNAP_DEBUG === 'true') {
        console.log(chalk.cyanBright(`[DEBUG] ${message}`));
    }
}

// --- CONFIGURATION ---
function getConfig() {
    const cfgPath = process.env.SNAP_CONFIG_PATH;
    let cfg = {};
    if (cfgPath && fs.existsSync(cfgPath)) {
        try { cfg = JSON.parse(fs.readFileSync(cfgPath, 'utf8')); }
        catch { logDebug('.snapconfig is corrupt'); }
    }
    cfg.options = cfg.options || {};
    return cfg;
}

function updateConfigOption(key, val) {
    const cfgPath = process.env.SNAP_CONFIG_PATH;
    if (!cfgPath) {
        console.error(chalk.red('[snap] Error: SNAP_CONFIG_PATH not set'));
        process.exit(1);
    }
    const cfg = fs.existsSync(cfgPath) ? JSON.parse(fs.readFileSync(cfgPath, 'utf8')) : {};
    
    if (key === 'backup_root') {
        cfg[key] = val;
    } else {
        cfg.options = cfg.options || {};
        cfg.options[key] = val;
    }

    fs.writeFileSync(cfgPath, JSON.stringify(cfg, null, 2));
}

// --- GENERIC COMMAND RUNNER ---
function runCommand(cmd, description, options = {}) {
    logDebug(`> Running Shell Command: ${description || cmd}`);
    try {
        const execOptions = {
            encoding: 'utf8',
            stdio: 'pipe',
            maxBuffer: 1024 * 1024 * 100, // 100 MB
            ...options
        };
        const output = execSync(cmd, execOptions);
        const cleanOutput = output.trim();
        logDebug(`< Clean Output:\n---\n${cleanOutput || '(empty)'}\n---`);
        return cleanOutput;
    } catch (error) {
        const errorMessage = error.stderr || error.stdout || 'Unknown command error.';
        const err = new Error(`Command failed: ${cmd}\n${errorMessage}`);
        err.stderr = errorMessage;
        logDebug(`! FAILED with exit code ${error.status}:\n${errorMessage}`);
        throw err;
    }
}

// --- EXTERNAL METADATA HANDLING (HIDDEN FILES & EMPTY DIRS) ---
function getProjectId() {
    const projectPath = process.cwd();
    const projectName = path.basename(projectPath);
    const projectHash = crypto.createHash('sha1').update(projectPath).digest('hex').substring(0, 8);
    const sanitizedName = projectName.replace(/[<>:"/\\|?*]/g, '_');
    return `${sanitizedName}_[${projectHash}]`;
}

function getExternalMetadataPath(commitId, type) {
    const config = getConfig();
    if (!config.backup_root) {
        logDebug('Cannot get metadata path: backup_root is not configured.');
        return null;
    }
    const projectId = getProjectId();
    const metadataDir = path.join(config.backup_root, projectId);
    return {
        dir: metadataDir,
        file: path.join(metadataDir, `${commitId}.${type}.json`)
    };
}

function _getEmptyDirs(rootDir) {
    logDebug(`Scanning for empty directories in: ${rootDir}`);
    const emptyDirs = [];
    const allDirs = [];
    const rootPath = path.resolve(rootDir);
    const ignoredDirs = new Set(['.git', 'node_modules']);
    
    function findAllDirs(currentPath) {
        try {
            const entries = fs.readdirSync(currentPath, { withFileTypes: true });
            for (const entry of entries) {
                if (ignoredDirs.has(entry.name)) continue;
                if (entry.isDirectory()) {
                    const dirPath = path.join(currentPath, entry.name);
                    allDirs.push(dirPath);
                    findAllDirs(dirPath);
                }
            }
        } catch (e) {
            logDebug(`Cannot read directory to find subdirs: ${currentPath}`);
        }
    }
    
    findAllDirs(rootPath);

    for (const dir of allDirs) {
        try {
            if (fs.readdirSync(dir).length === 0) {
                emptyDirs.push(path.relative(rootPath, dir).replace(/\\/g, '/'));
            }
        } catch (e) {
            logDebug(`Cannot read directory to check for emptiness: ${dir}`);
        }
    }
    
    logDebug(`Total found ${emptyDirs.length} empty directories.`);
    return emptyDirs.sort();
}

async function _getHiddenPathsOptimized(rootDir) {
    logDebug(`Scanning for hidden attributes in: ${rootDir}`);
    const hiddenPaths = [];
    const cmd = `dir /s /b /a:h "${rootDir}"`;
    try {
        const output = execSync(cmd, { maxBuffer: 10 * 1024 * 1024, encoding: 'utf8' });
        const lines = output.split(/\r?\n/);
        for (const line of lines) {
            const fullPath = line.trim();
            if (fullPath && fs.existsSync(fullPath)) {
                const relativePath = path.relative(rootDir, fullPath).replace(/\\/g, '/');
                if (!relativePath.startsWith('.git/')) {
                    hiddenPaths.push(relativePath);
                }
            }
        }
    } catch (error) {
        logDebug(`'dir' command for hidden files returned no results or an error (this is often normal).`);
    }
    logDebug(`Total found ${hiddenPaths.length} hidden items.`);
    return hiddenPaths.sort();
}

async function saveExternalMetadata(commitId) {
    // Hidden files
    const hiddenPathsData = getExternalMetadataPath(commitId, 'hidden');
    if (hiddenPathsData) {
        console.log(chalk.cyan('[snap] Scanning for hidden file attributes...'));
        const hiddenPaths = await _getHiddenPathsOptimized(process.cwd());
        if (hiddenPaths.length > 0) {
            logDebug(`Saving metadata for ${hiddenPaths.length} hidden items to ${hiddenPathsData.file}`);
            fs.mkdirSync(hiddenPathsData.dir, { recursive: true });
            fs.writeFileSync(hiddenPathsData.file, JSON.stringify(hiddenPaths, null, 2));
        } else if (fs.existsSync(hiddenPathsData.file)) {
            fs.unlinkSync(hiddenPathsData.file);
        }
    }

    // Empty directories
    const emptyDirsData = getExternalMetadataPath(commitId, 'emptydirs');
    if (emptyDirsData) {
        console.log(chalk.cyan('[snap] Scanning for empty directories...'));
        const emptyDirs = _getEmptyDirs(process.cwd());
        if (emptyDirs.length > 0) {
            logDebug(`Saving metadata for ${emptyDirs.length} empty directories to ${emptyDirsData.file}`);
            fs.mkdirSync(emptyDirsData.dir, { recursive: true });
            fs.writeFileSync(emptyDirsData.file, JSON.stringify(emptyDirs, null, 2));
        } else if (fs.existsSync(emptyDirsData.file)) {
            fs.unlinkSync(emptyDirsData.file);
        }
    }
}

function loadExternalMetadata(commitId, type) {
    if (!commitId || !type) return [];
    const paths = getExternalMetadataPath(commitId, type);
    if (!paths || !fs.existsSync(paths.file)) return [];
    try {
        return JSON.parse(fs.readFileSync(paths.file, 'utf8'));
    } catch (e) { return []; }
}

function deleteExternalMetadata(commitId) {
    if (!commitId) return;
    const types = ['hidden', 'emptydirs'];
    let cleanedSomething = false;

    for (const type of types) {
        const paths = getExternalMetadataPath(commitId, type);
        if (paths && fs.existsSync(paths.file)) {
            logDebug(`Deleting metadata file: ${paths.file}`);
            try {
                fs.unlinkSync(paths.file);
                cleanedSomething = true;
            } catch (e) {

    try {
        const command = 'git tag -l --sort=-taggerdate --format="%(refname:short)\t%(*objectname)\t%(taggerdate:iso8601)\t%(contents:subject)"';
        const tagOutput = runCommand(command);
        
        if (!tagOutput) return [];

        const lines = tagOutput.split('\n').filter(line => line.trim() !== '');
        const snapshots = lines.map(line => {
            const [tag, fullId, timestamp, description] = line.split('\t');
            if (!tag || !fullId) return null;
            return {
                fullId,
                id: fullId.substring(0, 7),
                tag,
                label: tag,
                description: (description || '').trim(),
                timestamp,
            };
        }).filter(Boolean);
        
        snapshots.sort((a, b) => new Date(b.timestamp) - new Date(a.timestamp));
        return snapshots;
    } catch (e) {
        logDebug(`! ERROR during getSnapshots: ${e.message}. Returning empty array.`);
        return [];
    }

function formatTimestamp(iso) {
    if (!iso) return '';
    const d = new Date(iso);
    if (isNaN(d)) return 'Invalid date';
    const z = n => n.toString().padStart(2, '0');
    return `${z(d.getDate())}.${z(d.getMonth() + 1)}.${d.getFullYear()} ${z(d.getHours())}:${z(d.getMinutes())}`;
}

    if (!snaps || snaps.length === 0) {
        outputLines.push(chalk.yellow('\n  No snapshots found. Use "snap new <label>" to create one.'));
        return outputLines.join('\n');
    }

    outputLines.push('');
    const idCol = 9;
    const labelCol = 16;
    const descCol = 28;
    const timeCol = 17;
  
    let header = '  ';
    let separator = '  ';

        header += 'ID'.padEnd(idCol);
        separator += '-'.repeat(idCol - 2).padEnd(idCol);

    header += 'Label'.padEnd(labelCol) + 'Description'.padEnd(descCol) + 'Timestamp';
    separator += '-'.repeat(labelCol - 2).padEnd(labelCol) +
                 '-'.repeat(descCol - 2).padEnd(descCol) +
                 '-'.repeat(timeCol - 1).padEnd(timeCol);
               
    outputLines.push(chalk.bold(header));
    outputLines.push(chalk.bold(separator));
  
    const activeCommitId = await getActiveCommitFull(); 
    logDebug(`Active commit ID for list comparison is: "${activeCommitId}"`);

    for (const snap of snaps) {
        const isActive = activeCommitId && activeCommitId === snap.fullId;
        const labelStr = (snap.tag || '').substring(0, labelCol - 2).padEnd(labelCol);
        const descStr  = (snap.description || '').substring(0, descCol - 2).padEnd(descCol);
        const timeStr  = formatTimestamp(snap.timestamp).padEnd(timeCol);
        
        let line = '  ';
        if (showIds) {
            line += (snap.id || '').substring(0, idCol - 2).padEnd(idCol);
        }
        line += labelStr + descStr + timeStr;
    
        if (isActive) {
            line += chalk.green.bold('   (active)');
        }
        outputLines.push(line);
    }
    return outputLines.join('\n');

# 📦 Snap (The Rust Edition)

**A truly native, blazing-fast snapshot tool for Windows developers. Create efficient, versioned backups of your project folders with full support for hidden file attributes.**

`snap` is a command-line utility that provides a simple way to capture point-in-time states of your projects. It uses the power and reliability of Git for its core operations but provides a simplified, focused workflow. It's perfect for quick, local backups before a major refactor, for archiving project milestones, or for any situation where you need a reliable "undo" button for your entire directory.

## From Node.js to Native: The Quest for Speed

This project began its life as a Node.js application, packaged into an `.exe` for convenience. While functional, it had a noticeable startup delay inherent to the Node.js runtime. For a tool designed to be a quick, seamless part of a developer's workflow, this was a critical friction point.

The desire for **instantaneous, native-level performance** led to a complete rewrite in **Rust**. By compiling directly to a native binary, `snap` now launches as fast as `git` itself, eliminating all runtime overhead. This new version retains 100% of the original's features while providing the speed and responsiveness a command-line power tool deserves.

## ✨ Key Features

*   **🚀 Truly Native Performance**: Rewritten in Rust for instantaneous startup and execution. Feels as fast and responsive as `git` itself.
*   ** dependable Git Core**: Leverages the rock-solid foundation of `git.exe` for all file storage and versioning. Snapshots are lightweight, annotated Git tags.
*   **⚡️ Efficient Storage**: Benefits from Git's mature Content-Addressable Storage model. Identical files across hundreds of snapshots consume the space of just one file.
*   **🎯 Smart Restores**: `snap restore` intelligently checks for uncommitted local changes and prompts you to prevent accidental data loss. It also restores your project files and synchronizes hidden file attributes.
*   **🪟 Windows Native**: Fully aware of Windows file attributes. It correctly preserves and restores the **Hidden** status of files and folders using an external metadata store.
*   **✍️ Rich Metadata**: Snapshots are tagged with a simple `label` and a more detailed `description`, which are stored directly in Git's annotated tags.

*   **🚫 Zero Dependencies**: The compiled `snap.exe` is a single, standalone binary. No need for Node.js, Rust, or any other runtime on the user's machine.

## ⚙️ How It Works

`snap` manages your project as a standard Git repository but uses a simple, robust interaction model.

1.  **The Project Repository (`.git`)**
    When you run `snap init`, your project folder becomes a standard Git repository. `snap` then acts as a user-friendly wrapper around `git.exe`.

2.  **The External Metadata Store**
    Because Git doesn't track the "Hidden" attribute on Windows, `snap` saves this information in a central `backup_root` directory you configure. For each snapshot, it creates a simple JSON file listing all hidden paths.

**Visualized Structure:**

```text
// Your project folder becomes a Git repository:
D:/Projects/MyUnityGame/
└── .git/

// A separate, central location stores the hidden attribute metadata:
D:/Backups/Snap_Metadata/
└── MyUnityGame_[a4e8b91c]/  <-- Human-readable, unique ID for your project
    ├── 0a1b2c...d3e4.json   <-- Hidden paths for commit 0a1b...
    └── 5f6a7b...c8d9.json   <-- Hidden paths for commit 5f6a...
```

This design keeps your project folder clean and standard while ensuring no data is lost.

## 🚀 Usage

All commands are run from within your project's directory.

### 1. `snap init`
Initializes the current folder as a `snap` repository. **Must be run once per project.**

```cmd
D:\Projects\my-app>snap init

[snap] Set backup_root to: D:/Snap_Metadata
[snap] Initialized empty snap repository in D:\Projects\my-app
```

### 2. `snap new <label> [description...]`
Creates a snapshot of the current project state.

```cmd
D:\Projects\my-app>snap new v1.0 Initial release

[snap] Step 1/4: Staging all files...
[snap] Step 2/4: Creating the commit...
[snap] Step 3/4: Saving file attribute metadata...
[snap] Step 4/4: Creating the annotated snapshot tag...

[snap] New snapshot created: [a1b2c3d] v1.0
```

### 3. `snap list`
Lists all available snapshots in a compact view, showing the active one.

```cmd
D:\Projects\my-app>snap list

[snap] Snapshots for "my-app":

  v1.0            Initial release             15.06.2025 17:45
```

### 4. `snap diff <snapshot-A> <snapshot-B>`
Compares two snapshots and shows a list of changes.

```cmd
D:\Projects\my-app>snap diff v1.0 v1.1

[snap] Comparing snapshots a1b2c3d ("v1.0") ➜ f4e5d6c ("v1.1"):

  ! .env (visibility changed)

[snap] Summary: 1 added, 1 deleted, 1 modified, 1 visibility change

Restores the project to a previous state. If run without arguments, it displays an interactive menu.

```cmd
D:\Projects\my-app>snap restore v1.0

[snap] Synchronizing hidden file attributes...

```cmd

use crate::cli::DeleteArgs;
use crate::config::{ensure_config_exists, load_config};
use crate::utils::{
    ask_yes_no, delete_external_attributes, find_snapshot, format_snapshot_line, get_snapshots,
    run_command, Snapshot,
};
use anyhow::{anyhow, Result};
use colored::*;
use inquire::Select;

pub fn execute(args: DeleteArgs) -> Result<()> {
    ensure_config_exists()?;
    let config = load_config()?;

    let snapshots = get_snapshots()?;
    if snapshots.is_empty() {
        return Err(anyhow!("No snapshots found to delete."));
    }

    // Determine which snapshot to delete, either from args or an interactive menu
    let snapshot_to_delete = match args.id_or_label {
        Some(key) => find_snapshot(&snapshots, &key)
            .cloned()
            .ok_or_else(|| anyhow!("Snapshot \"{}\" not found.", key)),
        None => {
            let choices: Vec<String> = snapshots
                .iter()
                .map(|s| format_snapshot_line(s, config.options.show_ids))
                .collect();
            let choice = Select::new("Select snapshot to delete:", choices).prompt()?;
            snapshots
                .iter()
                .find(|s| format_snapshot_line(s, config.options.show_ids) == choice)
                .cloned()
                .ok_or_else(|| anyhow!("Could not find selected snapshot."))
        }
    }?;

    println!("\n{}", "[snap] You are about to delete snapshot:".cyan());
    println!("  Label: {}", snapshot_to_delete.tag);

    if config.options.confirm_command {
        let proceed = ask_yes_no(
            "[snap] WARNING: This will permanently delete the snapshot tag. Continue?",
            false,
        )?;
        if !proceed {
            println!("{}", "[snap] Deletion cancelled.".yellow());
            return Ok(());
        }
    }

    println!("{}", format!("[snap] Deleting tag \"{}\"...", snapshot_to_delete.tag).cyan());
    run_command(&format!("git tag -d {}", snapshot_to_delete.tag), None)?;

    delete_external_attributes(&snapshot_to_delete.full_id)?;

    println!("{}", format!("[snap] Snapshot \"{}\" deleted successfully.", snapshot_to_delete.tag).green());
    Ok(())
}

use crate::cli::DiffArgs;
use crate::config::ensure_config_exists;
use crate::utils::{find_snapshot, get_snapshots, load_external_metadata};
use anyhow::{anyhow, Result};
use colored::*;
use std::collections::{HashMap, HashSet};

pub fn execute(args: DiffArgs) -> Result<()> {
    ensure_config_exists()?;

    let snapshots = get_snapshots()?;
    let snapshot_a = find_snapshot(&snapshots, &args.snapshot_a)
        .ok_or_else(|| anyhow!("Could not find snapshot '{}'", args.snapshot_a))?;
    let snapshot_b = find_snapshot(&snapshots, &args.snapshot_b)
        .ok_or_else(|| anyhow!("Could not find snapshot '{}'", args.snapshot_b))?;

    println!(
        "\n{} {} (\"{}\") ➜ {} (\"{}\"):\n",
        "[snap] Comparing snapshots".cyan(),
        snapshot_a.id.yellow(),
        snapshot_a.tag,
        snapshot_b.id.yellow(),
        snapshot_b.tag
    );

    let hidden_a: HashSet<_> = load_external_metadata(&snapshot_a.full_id)?.into_iter().collect();
    let hidden_b: HashSet<_> = load_external_metadata(&snapshot_b.full_id)?.into_iter().collect();
    
    let diff_output = crate::utils::run_command(
        &format!("git diff --name-status {} {}", snapshot_a.full_id, snapshot_b.full_id),
        None,
    )?;

    let mut changes = HashMap::from([
        ("added", 0), ("deleted", 0), ("modified", 0), ("renamed", 0), ("copied", 0), ("visibility", 0)
    ]);
    let mut files_in_diff = HashSet::new();

    for line in diff_output.lines() {
        let parts: Vec<&str> = line.split('\t').collect();
        let status = parts[0].chars().next().unwrap_or('?');
        let file_path = parts.get(1).copied().unwrap_or("");
        
        let (prefix, color, label) = match status {
            'A' => ("+", "green", "added"),
            'D' => ("-", "red", "deleted"),
            'M' => ("~", "yellow", "modified"),
            'R' => (">", "cyan", "renamed"),
            'C' => ("C", "cyan", "copied"),
            _ => ("?", "white", "unknown"),
        };

        println!("  {}", format!("{} {}", prefix, file_path).color(color));
        *changes.entry(label).or_insert(0) += 1;
        files_in_diff.insert(file_path.to_string());
    }

    let all_hidden_paths: HashSet<_> = hidden_a.union(&hidden_b).cloned().collect();

    for path in all_hidden_paths {
        if files_in_diff.contains(&path) { continue; }
        
        let is_hidden_a = hidden_a.contains(&path);
        let is_hidden_b = hidden_b.contains(&path);

        if is_hidden_a != is_hidden_b {
            println!("  {}", format!("! {} (visibility changed)", path).magenta());
            *changes.entry("visibility").or_insert(0) += 1;
        }
    }

    let summary_parts: Vec<String> = changes.iter()
        .filter(|(_, &count)| count > 0)
        .map(|(&label, &count)| {
            if label == "visibility" {
                format!("{} visibility {}", count, if count > 1 {"changes"} else {"change"})
            } else {
                format!("{} {}", count, label)
            }
        }).collect();

    if summary_parts.is_empty() {
        println!("{}", "[snap] The two snapshots are identical. No differences found.".green());
    } else {
        println!("\n{} {}", "[snap] Summary:".cyan(), summary_parts.join(", "));
    }

    Ok(())
}

pub fn execute(args: EditArgs) -> Result<()> {
    ensure_config_exists()?;
    let config = load_config()?;

    let snapshots = get_snapshots()?;
    if snapshots.is_empty() {
        return Err(anyhow!("No snapshots found to edit."));

    // Determine which snapshot to edit, either from args or an interactive menu
    let snapshot_to_edit = match args.id_or_label {
        Some(key) => find_snapshot(&snapshots, &key)
            .cloned()
            .ok_or_else(|| anyhow!("Snapshot \"{}\" not found.", key)),
        None => {
            let choices: Vec<String> = snapshots
                .iter()
                .map(|s| format_snapshot_line(s, config.options.show_ids))
                .collect();
            let choice = Select::new("Select snapshot to edit:", choices).prompt()?;
            snapshots
                .iter()
                .find(|s| format_snapshot_line(s, config.options.show_ids) == choice)
                .cloned()
                .ok_or_else(|| anyhow!("Could not find selected snapshot."))
        }
    }?;

    println!("\n{} \"{}\":", "[snap] Editing snapshot".cyan(), snapshot_to_edit.tag);

    let new_label = Text::new("Enter new label (tag name):")
        .with_initial_value(&snapshot_to_edit.tag)
        .with_validator(inquire::required!("Label cannot be empty."))
        .prompt()?;
    
    let new_description = Text::new("Enter new description:")
        .with_initial_value(&snapshot_to_edit.description)
        .prompt()?;

    let new_tag_name = sanitize_tag_name(&new_label);

    if new_tag_name == snapshot_to_edit.tag && new_description == snapshot_to_edit.description {
        println!("{}", "[snap] No changes detected. Edit cancelled.".yellow());
        return Ok(());
    }

    // Fail fast if the new tag name already exists (and it's not the same tag)
    if new_tag_name != snapshot_to_edit.tag && snapshots.iter().any(|s| s.tag == new_tag_name) {
        return Err(anyhow!("A snapshot with the label \"{}\" already exists.", new_tag_name));
    }

    println!("\n{}", "[snap] Applying changes... This will replace the old tag.".yellow());

    // Delete the old tag
    run_command(&format!("git tag -d {}", snapshot_to_edit.tag), None)?;

    // Create the new tag pointing to the same commit
    let tag_cmd = format!("git tag -a \"{}\" -F - {}", new_tag_name, snapshot_to_edit.full_id);
    run_command(&tag_cmd, Some(&new_description))?;

    println!("\n{}", format!("[snap] Snapshot successfully updated to \"{}\".", new_tag_name).green());

    Ok(())
}

// A small helper trait to make padding strings to a fixed width easier.
trait PadStr {
    fn pad_to_width(&self, width: usize) -> String;
}

// Implement the trait for string slices. This will truncate long strings and pad short ones.
impl PadStr for &str {
    fn pad_to_width(&self, width: usize) -> String {
        let mut s = self.to_string();
        // Truncate if the string is longer than the available space
        if s.len() > width.saturating_sub(2) {
            s.truncate(width.saturating_sub(2));
        }
        // Pad with spaces to fill the column width
        format!("{:<width$}", s, width = width)
    }
}

pub fn execute(_args: ListArgs) -> Result<()> {
    // Ensure we are in an initialized snap project
    ensure_config_exists()?;
    let config = load_config()?;
    let snapshots = get_snapshots()?;
    let active_commit = get_active_commit_full()?.unwrap_or_default();

    // Get the current directory name for the header
    let project_name = Path::new(".")
        .canonicalize()?
        .file_name()
        .and_then(|s| s.to_str())
        .unwrap_or("current project");

    println!("\n{} \"{}\":", "[snap] Snapshots for".cyan(), project_name);

    if snapshots.is_empty() {
        println!("\n  {}", "No snapshots found. Use \"snap new <label>\" to create one.".yellow());
        return Ok(());
    }

    // Define column widths, matching the original Node.js version
    let show_ids = config.options.show_ids;
    let id_w = if show_ids { 9 } else { 0 };
    let label_w = 16;
    let desc_w = 28;
    let time_w = 17;

    // Build the header string
    let mut header = "  ".to_string();
    let mut separator = "  ".to_string();

    if show_ids {
        header.push_str(&"ID".pad_to_width(id_w));
        separator.push_str(&"-".repeat(id_w - 2).pad_to_width(id_w));
    }
    header.push_str(&"Label".pad_to_width(label_w));
    header.push_str(&"Description".pad_to_width(desc_w));
    header.push_str("Timestamp");

    separator.push_str(&"-".repeat(label_w - 2).pad_to_width(label_w));
    separator.push_str(&"-".repeat(desc_w - 2).pad_to_width(desc_w));
    separator.push_str(&"-".repeat(time_w - 1).pad_to_width(time_w));
    
    println!("\n{}", header.bold());
    println!("{}", separator.bold());

    // Iterate through snapshots and print each formatted row
    for snap in snapshots {
        let is_active = !active_commit.is_empty() && snap.full_id == active_commit;
        let mut line = "  ".to_string();

        if show_ids {
            line.push_str(&snap.id.as_str().pad_to_width(id_w));
        }
        line.push_str(&snap.tag.as_str().pad_to_width(label_w));
        line.push_str(&snap.description.as_str().pad_to_width(desc_w));
        line.push_str(&format_timestamp(&snap.timestamp).as_str().pad_to_width(time_w));
        
        if is_active {
            line.push_str(&format!("   {}", "(active)".green().bold()));
        }
        println!("{}", line);
    }
    
    Ok(())
}

pub fn execute(_args: OptionsArgs) -> Result<()> {
    let mut config = load_config()?;

    let options_map = vec![
        ("showIds", "Controls if IDs are shown in lists"),
        ("confirm_command", "Asks for y/N on destructive actions"),
    ];

    let display_options: Vec<String> = options_map
        .iter()
        .map(|(key, desc)| {
            let current_value = match *key {
                "showIds" => config.options.show_ids,
                "confirm_command" => config.options.confirm_command,
                _ => false,
            };
            format!("{:<18} - {} (current: {})", key, desc, current_value.to_string().cyan())

        .iter()
        .find(|(_, _)| choice.contains(&format!("{} -", _ .0)))

        .unwrap();

        _ => unreachable!(),

use crate::config::{ensure_config_exists, load_config};
use crate::utils::{

use std::env;
use std::path::Path;

pub fn execute(args: RestoreArgs) -> Result<()> {
    ensure_config_exists()?;
    let config = load_config()?;

    let source_commit_id = get_active_commit_full()?.unwrap_or_default();

    if check_dirty()? {
        println!("\n{}", "[snap] WARNING: Your project has uncommitted changes.".yellow());
        let proceed = ask_yes_no(
            "To restore a snapshot, all local changes must be discarded. Continue?",
            false,
        )?;
        if !proceed {
            println!("{}", "[snap] Restore cancelled.".yellow());
            return Ok(());
        }
        println!("{}", "[snap] Discarding all local changes...".cyan());
        run_command("git reset --hard HEAD", None)?;
        run_command("git clean -fd", None)?;
        println!("{}\n", "[snap] Workspace is now clean. Proceeding with restore.".green());
    }

    let snapshots = get_snapshots()?;
    if snapshots.is_empty() {
        return Err(anyhow!("No snapshots found to restore."));
    }

    // Determine which snapshot to restore, either from args or an interactive menu
    let snapshot_to_restore = match args.id_or_label {
        Some(key) => find_snapshot(&snapshots, &key)
            .cloned() // Clone the snapshot data
            .ok_or_else(|| anyhow!("Snapshot \"{}\" not found.", key)),
        None => {
            let choices: Vec<String> = snapshots
                .iter()
                .map(|s| format_snapshot_line(s, config.options.show_ids))
                .collect();
            let choice = Select::new("Select snapshot to restore:", choices).prompt()?;
            // Find the original snapshot struct from the formatted string
            snapshots
                .iter()
                .find(|s| format_snapshot_line(s, config.options.show_ids) == choice)
                .cloned()
                .ok_or_else(|| anyhow!("Could not find selected snapshot."))
        }
    }?;

    println!("\n{} \"{}\"...", "[snap] Restoring project files for snapshot".cyan(), snapshot_to_restore.tag);
    run_command(&format!("git checkout --force {}", snapshot_to_restore.tag), None)?;

    // --- Attribute Synchronization ---
    println!("{}", "[snap] Synchronizing hidden file attributes...".cyan());
    
    // Use HashSets for efficient lookups
    let source_hidden: HashSet<_> = load_external_metadata(&source_commit_id)?.into_iter().collect();
    let target_hidden: HashSet<_> = load_external_metadata(&snapshot_to_restore.full_id)?.into_iter().collect();

    let cwd = env::current_dir()?;

    // Un-hide files that were hidden before but shouldn't be now
    for path_str in source_hidden.difference(&target_hidden) {
        let full_path = cwd.join(path_str);
        if full_path.exists() {
            // The `attrib -H` command in Windows makes a file visible
            let _ = run_command(&format!("attrib -H \"{}\"", full_path.to_string_lossy()), None);
        }
    }

    // Hide files that should be hidden in the target snapshot
    for path_str in &target_hidden {
        let full_path = cwd.join(path_str);
        if full_path.exists() {
            // The `attrib +H` command in Windows makes a file hidden
            let _ = run_command(&format!("attrib +H \"{}\"", full_path.to_string_lossy()), None);
        }
    }

    println!("\n{}", "[snap] Restore complete. Your project is now at the state of this snapshot.".green());

    Ok(())
}

#[derive(Serialize, Deserialize, Debug, Clone)]
pub struct SnapConfig {
    pub options: Options,
    pub backup_root: String,

#[derive(Serialize, Deserialize, Debug, Clone)]
pub struct Options {
    #[serde(default = "default_bool_false")]
    pub show_ids: bool,
    #[serde(default = "default_bool_true")]
    pub confirm_command: bool,
}

fn default_bool_false() -> bool { false }
fn default_bool_true() -> bool { true }

impl Default for SnapConfig {
    fn default() -> Self {
        SnapConfig {
            options: Options {
                show_ids: false,
                confirm_command: true,
            },
            backup_root: String::new(),
        }
    }
}

pub fn get_config_path() -> Result<PathBuf> {
    let exe_path =
        std_ext::env::current_exe().context("Failed to get current executable path")?;
    let snap_dir = exe_path
        .parent()
        .ok_or_else(|| anyhow!("Failed to get parent directory of executable"))?;
    Ok(snap_dir.join(".snapconfig"))
}

pub fn load_config() -> Result<SnapConfig> {
    let config_path = get_config_path()?;
    if !config_path.exists() {
        return Ok(SnapConfig::default());
    }
    let content = fs::read_to_string(&config_path)
        .with_context(|| format!("Failed to read config file at {:?}", config_path))?;
    serde_json::from_str(&content).context("Failed to parse .snapconfig JSON")
}

pub fn save_config(config: &SnapConfig) -> Result<()> {
    let config_path = get_config_path()?;
    let content = serde_json::to_string_pretty(config)
        .context("Failed to serialize config to JSON")?;
    fs::write(&config_path, content)
        .with_context(|| format!("Failed to write config file to {:?}", config_path))
}

pub fn ensure_config_exists() -> Result<()> {
    let config_path = get_config_path()?;
    if !config_path.exists() {
        return Err(anyhow!(
            "Configuration file .snapconfig not found. Run 'snap init' to create one."
        ));
    }
    Ok(())
}

#[derive(Debug, Clone)]
pub struct Snapshot {
    pub full_id: String,
    pub id: String,

    pub description: String,
    pub timestamp: String,

// A direct translation of your `runCommand` function.
pub fn run_command(cmd_str: &str, input: Option<&str>) -> Result<String> {
    let mut parts = cmd_str.split_whitespace();
    let command = parts.next().ok_or_else(|| anyhow!("Empty command"))?;
    let args: Vec<&str> = parts.collect();

    let mut cmd = Command::new(command);
    cmd.args(&args);

    if input.is_some() {
        cmd.stdin(Stdio::piped());
    }
    cmd.stdout(Stdio::piped()).stderr(Stdio::piped());

    let mut child = cmd.spawn().with_context(|| format!("Failed to spawn command: {}", cmd_str))?;

    if let (Some(stdin), Some(input_data)) = (child.stdin.as_mut(), input) {
        stdin.write_all(input_data.as_bytes())?;
    }

    let output = child.wait_with_output()?;

    if !output.status.success() {
        let stderr = String::from_utf8_lossy(&output.stderr);
        return Err(anyhow!(
            "Command failed: {}\n---\n{}",
            cmd_str,
            stderr.trim()
        ));
    }

    Ok(String::from_utf8(output.stdout)?)
}

pub fn is_git_repo() -> bool {

## Idiomatic Rust Usage Improvements

Beyond updating crate versions, a key part of modernization is refactoring the code to use current **idiomatic Rust patterns**. After reviewing typical CLI code (in `main.rs`, `utils.rs`, `commands/init.rs`), here are recommendations:

* **Use `Result` in `main`:** If not already done, have `fn main() -> anyhow::Result<()>` (or other Error type) instead of manually handling errors inside main. This lets you use the `?` operator at the top level and will automatically print error messages and set a non-zero exit code if an error occurs. It greatly simplifies error propagation from your `init` command or others. For example:

  ```rust
  fn main() -> anyhow::Result<()> {
      commands::init::run()?;  // if this returns Err, the error prints and program exits
      Ok(())
  }
  ```

  This is more idiomatic in 2025 than using `std::process::exit(...)` or printing errors manually.

* **Leverage Anyhow’s Context:** When handling operations like file I/O, wrap them with context. For instance, in `init.rs` if you open a file or create a directory, use `.with_context(|| "descriptive message")` to enrich any error. This yields errors like “Failed to create config directory: Permission denied” instead of just “Permission denied”. This makes the CLI more user-friendly when things go wrong.

* **Clap Derive API:** Simplify argument parsing by using **derive macros** provided by Clap instead of manually constructing the CLI in code. For example, you can define:

  ```rust
  #[derive(clap::Parser)]
  struct CliArgs {
      #[arg(short, long, help = "Enable verbose output")]
      verbose: bool,
      #[command(subcommand)]
      command: Commands,
  }
  #[derive(clap::Subcommand)]
  enum Commands {
      Init { #[arg(help="path to config")] path: String },
      // ... other subcommands
  }
  ```

  Then in `main()` do `let args = CliArgs::parse();` – Clap will handle the rest. This approach, introduced in Clap 3 and polished in Clap 4, is more declarative and reduces boilerplate. It automatically generates help messages from struct field metadata and validates arguments, making the code in `main.rs` much cleaner.

  * **`if let` chains and `let-else`:** These make control flow more readable. For example, if in `init.rs` you had:

        // use cfg

        return Err(anyhow!("No config path found"));

    ```

  * **`match` ergonomics:** Use wildcard matches and ignore unused variables with `_` if needed, and prefer `matches!` macro for simple boolean conditions. For instance, `if matches!(mode, Mode::Fast)` reads nicely.

  * **Path management:** If combining paths, use the `/` operator from Rust 1.63 for `PathBuf` (e.g. `path / "subdir" / "file.txt"`).
